-----
### Git Bash Winpty 
-----
1. ```echo "alias docker='winpty docker'" >> ~/.bashrc``` 명령어 입력
   - Docker라는 명령어 대신 Winpty Docker가 실행되도록 함
   - 즉, Docker 명령어 Custom
    
2. 재실행 후, alias 입력
